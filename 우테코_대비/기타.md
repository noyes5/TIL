### 입력값 다시 받기

-

```
public class DataInputHandler<T> {
    private final Supplier<T> supplier;

    private final Consumer<IllegalArgumentException> exceptionHandler;

    public DataInputHandler(Supplier<T> supplier, Consumer<IllegalArgumentException> exceptionHandler) {
        this.supplier = supplier;
        this.exceptionHandler = exceptionHandler;
    }

    public T initialize() {
        T item;
        while (true) {
            try {
                item = supplier.get();
                return item;
            } catch (IllegalArgumentException ex) {
                exceptionHandler.accept(ex);
            }
        }
    }
}
```

```
DataInputHandler<String> initializer = new DataInputHandler<>(
        this::readFunction,
        OutputView::printErrorMessage
);
return initializer.initialize();
```
